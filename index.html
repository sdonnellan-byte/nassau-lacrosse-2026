🚀 Go Live in 15 Minutes

Follow these steps once and your site will be live at a real URL, updating in real time for anyone in the world.

1

Create a Free Firebase Account

Go to firebase.google.com and click Get Started. Sign in with any Google account (Gmail works).

2

Create a New Project

Click "Add project" → name it nassau-lacrosse → disable Google Analytics → click Create project. Wait about 10 seconds for it to finish.

3

Set Up the Realtime Database

In the left sidebar click Build → Realtime Database → Create Database. Choose United States as the region. When asked about security rules, choose "Start in test mode" → click Enable.

4

Get Your Config Keys

Click the ⚙️ gear icon next to "Project Overview" → Project settings. Scroll down to "Your apps" and click the </> Web icon. Give the app a nickname like nassau-lax → click Register app.

You'll see a block of code. Copy the 7 values inside firebaseConfig — you need all of them for the next step.

📋 The values look like: apiKey, authDomain, databaseURL, projectId, storageBucket, messagingSenderId, appId
5

Paste Your Keys Into the HTML File

Open nassau_lacrosse_2026.html in Notepad (Windows) or TextEdit (Mac). Find this section near the bottom of the file:

apiKey: "PASTE_YOUR_API_KEY_HERE",
authDomain: "PASTE_YOUR_AUTH_DOMAIN_HERE",
databaseURL: "PASTE_YOUR_DATABASE_URL_HERE",
... and so on for all 7 values

Replace each PASTE_YOUR_..._HERE with the real value from Firebase. Keep the quotes around each value. Save the file when done.

6

Set Firebase Security Rules

In Firebase Console, go to Realtime Database → Rules tab. Replace everything there with this and click Publish:

{
  "rules": {
    "ncgl2026": {
      ".read": true,
      ".write": true
    }
  }
}
7

Deploy to Netlify (Your Live URL)

a) Go to app.netlify.com/drop in your browser.
b) Drag and drop your updated nassau_lacrosse_2026.html file onto the page.
c) Netlify gives you a URL like https://lively-fox-abc123.netlify.app — that's your live site!
d) To get a better URL: click Site configuration → Site details → Change site name and rename it to something like nassau-lacrosse-2026.
Your final URL will be: https://nassau-lacrosse-2026.netlify.app

8

You're Live! 🎉

Share your Netlify URL with coaches, parents, and league officials. From now on:

  • Admin enters scores → all visitors see the update within 1–2 seconds
  • Coaches submit scores for review → appears in admin queue instantly
  • Anyone can view standings, schedules, and team pages from any device
💡 Updating the site later: If you get a new version of the HTML file from Claude, just paste your same Firebase config into the new file and drag it to Netlify again. Your data is safe in Firebase — nothing gets reset.

Both Firebase and Netlify are free for this use case. No credit card required.
